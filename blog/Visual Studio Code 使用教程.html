# Visual Studio Code 使用教程

  1. 比较好用的图标主题：vscode-icons,vscode-icons-mac
  2. 设置编译器的基本设置，可以在 '文件>首选项>设置>用户' 中进行设置。
  3. 快捷键的设置
  • Ctrl + c :复制
  • Ctrl + v :粘贴
  • Ctrl + x :剪切
  • Ctrl + f :查找
  • Ctrl + h :替换
  • Ctrl + s :保存
  • Ctrl + / :行注释
  • Shift + Alt + A:块注释
  • Ctrl + Shift + Enter:上方插入一行
  • Ctrl + Shift + F: 文件夹查找
  • Ctrl + Enter:下方插入一行
  • Alt + Shift + F: 格式化代码
  1. 将其他编译器的快捷键应用到VScode上
  • '文件>首选项>按键映射'
  1. 自定义快捷键
  • '文件>首选项>键盘快捷键方式'
  1. vscode使用git
  • 什么是git
    1: git是一个开源的分布式版本控制系统
    2: 可以有效、高速的处理从很小到非常大的项目版本管理。 3：git输入路径时，可以在不完全输入路径按Tab键进行路径自动补全。 4： vscode控制面板中的‘输出’面板可以查看对应git的操作。 5:侧栏导>分支,对更新的文件可以点击三点选择提交，推送，拉取的步骤。
  1. vscode常用插件推荐
    • CSS Peek: HTML与CSS关联(Ctrl + 鼠标移动到（链接、id、css）上面点击：可以直接跳转到css样式文件)
    • Prettier - Code formatter :代码格式化
    • Icon Fonts :图标集，如 Font Awesome、Glyphicons
    • Auto Rename Tag: 自动修改标签
    • HTML Boilerplate: HTML模板
    • Color Info: 颜色提示
    • Auto Close Tag:自动闭合标签
    • HTML CSS Support: html中css class的智能提示。
